# Network Filtering v23

## Regel

```text
Netzwerktyp bestimmt Verbindungstyp-Auswahl.
```

## Beispiele

```text
Netzwerktyp WLAN
→ nur WLAN-Verbindungstypen
```

```text
Netzwerktyp LAN
→ nur LAN/PoE-Verbindungstypen
```

```text
Netzwerktyp VPN
→ nur VPN
```

```text
Netzwerktyp Offline
→ nur Offline
```

## Schutz

Auch beim Speichern wird geprüft, ob Netzwerktyp und Verbindungstyp zusammenpassen.
