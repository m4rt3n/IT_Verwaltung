# Bedingte Logik

## Gerätetyp
PC, Notebook, Thin Client:
- CPU, RAM, Speicher, Software aktiv

Drucker:
- Druckertyp, Toner, Zählerstand aktiv
- Software optional

Access Point / Switch:
- PoE, Controller, Management-Netzwerk aktiv

Monitor:
- Netzwerk/Software nicht im Fokus

## Netzwerk
Adressart DHCP:
- IP-Adresse und DNS deaktiviert

Adressart Statisch:
- IP-Adresse und DNS Pflicht

Verbindung WLAN:
- Access Point und SSID aktiv

Verbindung LAN:
- Wanddose und Switch-Port aktiv

## Ticket
Status Gelöst:
- Ursache und Lösung Pflicht
- Knowledge-Erstellung möglich
