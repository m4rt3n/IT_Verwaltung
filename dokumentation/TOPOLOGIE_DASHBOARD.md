# Topologie-Dashboard

Die Topologie-Ansicht zeigt Assets organigrammartig wie eine Netzwerk-Topologie.

## Ebenen

```text
Internet / WAN
→ Gateway
→ Switch / Server
→ Access Points / Drucker
→ Clients
```

## Farben

```text
Gateway = Violett
Switch/Netzwerk = Blau
Access Point = Türkis/Grün
Drucker = Orange
Clients = Grün
Defekt = Rot
```

## Interaktion

Asset-Knoten sind klickbar und öffnen den Asset-Tab.
