# Schnellstart

## Web-App starten

1. `start.bat` ausfuehren.
2. Windows-UAC bestaetigen.
3. Die angezeigte lokale URL im Browser oeffnen.
4. Im Dashboard pruefen, ob `CSV Backend aktiv` angezeigt wird.

## Erstes Geraet erfassen

1. `+ Neues Geraet erfassen` oeffnen.
2. Geraetetyp, Standort, Raum, Status und Nutzer pflegen.
3. Hardware- und Netzwerkdaten ergaenzen.
4. Pflichtfelder pruefen.
5. Final speichern und danach die Asset-Detailseite kontrollieren.

## Scanner starten

1. Web-App ueber `start.bat` mit Administratorrechten starten.
2. Admin Panel oeffnen.
3. Passenden Scanner waehlen: Hardware, Standardsoftware oder Full-Software.
4. Nach dem Scan Seite neu laden und Daten pruefen.

## Speichern und Backup

- Manuelle CSV-Speicherung liegt im Admin Panel.
- Vor dem Speichern erstellt der Server ein Backup unter `web_ui/backups/`.
- Produktive CSV-Dateien liegen unter `web_ui/data/`.
