# Animiertes Dashboard

## Ziel

Das Dashboard zeigt nicht nur Zahlen, sondern eine grafische Beziehungskarte.

## Darstellung

```text
IT / Assets
 ├── Asset 1
 │    ├── Hardware
 │    ├── Software
 │    ├── Netzwerk
 │    ├── Tickets
 │    └── Notizen
 └── Asset 2
      └── ...
```

## Farben

```text
Asset = Blau
Hardware = Grün
Software = Violett
Netzwerk = Hellblau
Tickets = Gelb
Notizen = Orange
```

## Interaktion

Asset-Knoten sind klickbar und öffnen den Assets-Tab mit dem gewählten Gerät.
