# Referenzierte Erstellung

## Grundregel

Ein vollständiges neues Gerät wird weiterhin über den Wizard erstellt:

```text
+ Neues Gerät erfassen
```

Dabei entstehen automatisch:

```text
Asset
Hardware
Netzwerk
Software
Notiz
```

## Einzelmodule

Die Tabs Hardware, Netzwerk, Software, Tickets und Notizen legen keine losen Einträge mehr an.

Stattdessen gilt:

```text
+ Hardware zu Asset
+ Netzwerk zu Asset
+ Software zu Asset
+ Ticket zu Asset
+ Notiz zu Asset
```

## Vorteil

- keine Hardware ohne Asset
- keine Netzwerkdaten ohne Gerät
- bessere Auswertung
- saubere Verknüpfung über Asset-ID
- weniger Dubletten
