# Hersteller-Abhängigkeiten

## Ziel

Im Geräte-Wizard wird der Hersteller abhängig vom Gerätetyp gefiltert.

Beispiele:

```text
Gerätetyp = Drucker
→ Kyocera, Canon, Brother, HP, Ricoh
```

```text
Gerätetyp = Notebook
→ Dell, Lenovo, HP, Fujitsu
```

```text
Gerätetyp = Access Point
→ Ubiquiti
```

## Neue Anlage

Im Wizard gibt es:

```text
+ Hersteller
```

Damit kann direkt ein Hersteller für den aktuell gewählten Gerätetyp ergänzt werden.

## Modellserien

Zusätzlich gibt es abhängig von Hersteller + Gerätetyp Modellserien.

Beispiel Fujitsu:

```text
Fujitsu + Notebook → Lifebook
Fujitsu + PC → Esprimo
Fujitsu + Workstation → Celsius
Fujitsu + Monitor → B-Line / P-Line
```

## Hardware-Neuanlage

Der Button „+ Hardware zu Asset“ wurde entfernt.

Hardware entsteht im normalen Prozess über:

```text
+ Neues Gerät erfassen
```

Danach kann Hardware im Hardware-Tab bearbeitet werden.
