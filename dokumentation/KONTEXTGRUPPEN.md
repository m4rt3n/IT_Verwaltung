# Kontextgruppen

## Hardware

Farbe: Grün

Bedeutung:
- Gerätehersteller
- Modell / Modellserie
- Seriennummer
- Inventarnummer
- CPU / RAM / Speicher
- Monitor / Dockingstation
- Garantie

## Software

Farbe: Violett

Bedeutung:
- Softwarehersteller
- Software / Anwendung
- Version
- Lizenzstatus
- Update-Status
- Kritikalität

Wichtig:
`Softwarehersteller` ist nicht dasselbe wie `Gerätehersteller`.

## Netzwerk

Farbe: Blau

Bedeutung:
- Netzwerktyp
- Adressart DHCP/Statisch
- Verbindungstyp LAN/WLAN
- IP / MAC / DNS
- VLAN
- Switch-Port / Wanddose
- Access Point / SSID

## Bedingte Logik

Software:
- Softwarehersteller steuert Software-Auswahl

Netzwerk:
- DHCP/Statisch steuert IP/DNS
- LAN/WLAN steuert Switch/Wanddose oder AP/SSID

Hardware:
- Gerätetyp steuert Hersteller/Modellserien im Wizard
