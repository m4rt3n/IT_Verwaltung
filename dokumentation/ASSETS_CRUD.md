# Assets bearbeiten und löschen

## Bearbeiten

Im Assets-Tab gibt es jetzt:

```text
Asset bearbeiten
```

Die Asset-ID bleibt fest.

## Löschen

Beim Löschen eines Assets werden abhängige Referenzen mit gelöscht:

```text
Hardware
Netzwerk
Software
Tickets
Notizen
```

Vor dem Löschen zeigt die Oberfläche die Anzahl der abhängigen Einträge an.
