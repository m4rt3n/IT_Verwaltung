# Safe UX Logic v22

## Netzwerk

### LAN
- Switch-Port Pflicht
- Wanddose empfohlen
- AP/SSID ausgeblendet

### WLAN
- Access Point Pflicht
- SSID Pflicht
- Switch-Port/Wanddose ausgeblendet

### VPN
- keine physische Infrastruktur
- Switch-Port/Wanddose/AP/SSID ausgeblendet

### Offline
- Netzwerkdaten werden nicht gepflegt

### DHCP
- IP/DNS werden geleert und deaktiviert

### Statisch
- IP/DNS Pflicht
- IPv4-Format wird geprüft

## Vorschau
Die Vorschau zeigt einen Prüfstatus mit Fehlern/Warnungen.
