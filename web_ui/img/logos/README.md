# Lokale Software-Logos

Diese SVG-Dateien werden in der Full-Software-Auswertung lokal genutzt.

Quelle der geladenen Marken-Icons:

- Simple Icons CDN: `https://cdn.jsdelivr.net/npm/simple-icons@latest/icons/`
- Projekt: `https://simpleicons.org/`

Hinweis:

- Die SVG-Dateien dienen nur der lokalen UI-Erkennung.
- Marken und Logos bleiben Eigentum der jeweiligen Markeninhaber.
- Wenn ein Produktlogo nicht verfuegbar ist, nutzt die Web-App automatisch ein kurzes Textkennzeichen aus `web_ui/config/software_classification.json`.
